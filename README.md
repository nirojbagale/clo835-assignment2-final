## Description

This project deploys a containerized web application and MySQL database
into a locally simulated single-node Kubernetes cluster using kind.

## Structure

- Namespaces
- Pods
- ReplicaSets
- Deployments
- Services

## How to Deploy

1. Create cluster using kind
2. Apply namespaces
3. Apply pods
4. Apply ReplicaSets
5. Apply Deployments
6. Apply Services

Example:
kubectl apply -f manifests/
